# frontend-project
Tech Stack Used

-> React (TypeScript) – component‑based UI

-> Tailwind CSS – utility‑first styling for pixel precision

-> Next.js (Pages Router) – framework & routing (bonus)

-> Framer Motion – smooth animations & transitions (bonus)

-> Modern ES6+ JavaScript

Key Features Implemented

-> Pixel‑perfect desktop UI matching Figma specs

-> Responsive layout constrained to desktop breakpoints

-> Hover states & micro‑interactions from prototype

-> Smooth animations using Framer Motion

-> Accessible semantic HTML (WCAG 2.1)

-> Keyboard navigable & focus‑visible states

-> Clean, reusable component architecture

Accessibility (WCAG 2.1)

-> Semantic HTML elements (nav, main, section, footer)

-> Proper heading hierarchy (h1 → h2 → h3)

-> Accessible color contrast

-> Focus indicators for interactive elements

-> ARIA labels where required

Assumptions Made

-> Desktop‑only implementation as specified (≥ 1280px)

-> Fonts and images exported directly from Figma

-> Colors, spacing, and typography strictly follow Figma tokens

-> No backend or API integration required
